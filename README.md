# 🛒 Superstore Sales Dashboard (Power BI)

## 📊 Overview
This project presents an interactive Power BI dashboard analyzing Superstore sales data. It helps in understanding sales performance, profit trends, and customer behavior across different regions.

## 🛠 Tools Used
- Power BI
- Excel (Dataset)

## 📈 Key KPIs
- Total Sales: 1.57M
- Total Profit: 175K
- Total Customers: 773
- Total Orders: 5901

## 📊 Features
- Region-wise filtering (Central, East, South, West)
- Sales by Payment Mode (COD, Online, Cards)
- Segment-wise analysis (Consumer, Corporate, Home Office)
- Category & Sub-category performance
- Monthly Sales & Profit trends
- Sales distribution using Map visualization

## 🔍 Key Insights
